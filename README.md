# terminal-emulator

```bash
$ yarn install

$ yarn dev

$ yarn build
$ yarn start
```

> ⚠️ This project uses a forked version of `express-ws` from ezracelli/express-ws.
>
> ```bash
> $ git clone git@github.com:ezracelli/express-ws.git
> $ cd express-ws && yarn link
> $ cd /path/to/terminal-emulator && yarn link "express-ws"
> ```
